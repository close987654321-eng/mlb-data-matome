# matome フィードバック蓄積ログ

このファイルは matome スキルが「使うほど育つ」ための学習メモ。
村山が個々のまとめ記事に出したダメ出し・好み・刺さったパターンをここに溜める。
スキルは**編集を始める前にここを読んでセルフレビュー**し（手順 Step 0）、**保存後に新しい学びを追記**する。

SKILL.md の R1〜R7 が「確定した編集ルール」なら、このログは**その手前のメモ**：
まだルール化していない細かな好み・1回限りかもしれない指摘を貯める場所。
同じ趣旨が2回来たら、それは R番号ルールに昇格させる合図（SKILL.md 側へ移して `【強】` 相当に）。

書き方のルール：
- 1項目 = 1行〜数行。具体的に。「この記事の○○がダメだった → 次はこうする」の形で。
- 競技固有なら `（mlb）`、シリーズ固有なら `（dodgers）` を添える。
- 「それ記録しなくていい／その記事だけの話」と言われたものは書かない。
- ルールに昇格させたら、ここからは消して SKILL.md の R番号に移す（二重管理しない）。
