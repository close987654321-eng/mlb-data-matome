# 海外の反応まとめ — エージェント共通ガイド（Codex 等）

このリポジトリは「**海外の反応まとめサイト**」です。**MLB・ボクシング・UFC** の海外掲示板
（主に Reddit）で盛り上がったスレッドを、現地ファンの生のコメントつきで日本語まとめ
（5ch まとめ風＝コメント翻訳中心）にして、英語が読めない日本のファンに届けます。

**運用ルールの唯一の正は [`CLAUDE.md`](./CLAUDE.md)**。このファイルには内容を複製しない
（過去に複製していて整合が崩れたため、ポインタに一本化した）。

- 技術スタック・ディレクトリ構成・データ形式・i18n・コード規約 → `CLAUDE.md`
- まとめ記事の編集ルール（抜粋・並べ方・翻訳・タイトル・要約 = R1〜R9） →
  [`.claude/skills/matome/SKILL.md`](./.claude/skills/matome/SKILL.md)
- データ取得・更新手順 → [`scripts/threads-update.md`](./scripts/threads-update.md)

## 絶対に守る制約（要約）

- コメントの捏造禁止。実在する発言だけを訳す。
- 全文転載禁止。**抜粋＋翻訳＋元スレ送客**の編集物にする（`sourceUrl` 必須）。
- Reddit データを AI/ML 学習に使わない。
- メディアファイルは `public/media/` 以外にコミットしない（詳細は CLAUDE.md §4.5）。
